API Documentation

Driver Model

Key Type Description
Id int The id of the driver
Name string Name of the driver
Age int Age of the driver
Nationality string Nationality of the driver
Image string URL to the driver's image

Get All Drivers

Endpoint: http://localhost:5266/api/Drivers

Returns all drivers in the database.

Get Driver by ID

Endpoint: http://localhost:5266/api/Drivers/2

Returns a single driver by their ID.

Get Driver by Name

Endpoint:http://localhost:5266/api/Drivers/title/Lewis%20Hamilton

Returns a driver by their name.
